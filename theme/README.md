# webpack
****https://webpack.js.org/****

#### install

````
mkdir <name-folder>
cd <name-folder>
npm init -y // create a package.json file, -y Generate it without having it ask any questions
npm install webpack webpack-cli --save-dev
````

#### webpack-cli

````
npm install --save-dev webpack-cli
````
